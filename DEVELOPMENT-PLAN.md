# DUSKFALL — Мастер-план развития

## Текущее состояние (v2.0)

### ✅ Реализовано
- AI Dungeon Master (LLM-driven) с строгими правилами атмосферы
- Мультиплеер (комнаты, socket.io синхронизация, ходы по инициативе)
- BG3-style создатель персонажа (12 классов, 9 рас, 10 происхождений, point-buy, предыстория)
- 120 талантов (10×12 классов) с 2-круговым деревом + ASI
- Способности (расовые, классовые, свитки) с быстрым применением
- Тактическая сетка 10×10 с targeting mode (атака, движение, способности)
- Система условий (10 типов) с иконками на токенах
- Преимущество/Помеха (5 источников, отмена)
- Спелл-слоты + короткий/долгий отдых
- AoE заклинания (круг/линия/конус, спасброски)
- Фланги и высокое положение
- Журнал квестов (автоматическое извлечение)
- Карта мира (BSP-генерация, туман войны)
- NPC + диалоги (торговля, проверки характеристик)
- Смена дня/ночи (4 фазы, тинты сцены)
- Погода (дождь/туман/гроза/снег, CSS-анимации)
- Случайные события (6 типов)
- Дерево навыков (2 круга + пререквизиты + ASI)
- Система экипировки (8 слотов, бонусы AC)
- Крафт (17 рецептов, 3 верстака)
- Анимации боя (движение, удар, вспышки, screen shake)
- Токены (портреты, рамки, HP-бар, ауры)
- Лог боя (фильтр, цвет, экспорт)
- Эффекты сетки (лут, ловушки, угрозы, AoE)
- Кастомизация UI (4 темы, 3 масштаба, сворачиваемые панели)
- Процедурное аудио (3 настроения музыки + 10 SFX + погода)
- Аккаунты + сохранения (3 слота, bcrypt, HMAC-cookies)
- Оптимизация БД (индексы, кеш снапшотов, pagination)
- Оптимизация LLM (обрезка контекста, кэш промптов, retry backoff, model routing)
- React.memo, виртуализация чата, dynamic polling
- Логирование, метрики, admin API
- Валидация, sanitize, rate limits, security headers
- CI workflow (lint + tsc + build)
- Генератор подземелий (5 биомов, BSP, ловушки, боссы, глубина)
- Мультипровайдер LLM (GLM → Gemini → OpenRouter → Ollama → z-ai-sdk)
