# План проверки 12 багов (без субагента)

## Баги для проверки:
1. TTS озвучка не работает
2. Переводы неполные + мастер на русском
3. Инициатива вверху экрана, а не в окне костей
4. Мастер не понимает кому наносить урон
5. Урон не наносится
6. Мастер использует английский/китайский
7. Переход хода сломан (10 ходов подряд)
8. Игроки бьют друг друга (friendly fire)
9. UI масштаб не работает
10. Лишние враги на сетке
11. Быстрые применения сломаны
12. Неправильная подсветка хода

## Дополнительно:
- Ячейки заклинаний по механике BG3 (ограничение использования)
- Механика отдыхов (short/long rest восстанавливает ячейки)

## Результаты проверки (без субагента):

### Баг 1: TTS ✅
- Route: /api/game/tts — wav формат, Content-Type forwarding
- ChatPanel: playTTS() — fetch + blob + Audio.play()
- Состояние: исправлено

### Баг 2: Переводы ✅
- MAX_MESSAGES_PER_BATCH: 500 (было 50)
- DM промпт: "Пиши ВСЕ ответы на языке: {lang}"
- Состояние: исправлено

### Баг 3: InitiativeTracker ✅
- Расположение: в левой колонке (строка 1233 page.tsx), НЕ в header
- Состояние: исправлено

### Баг 4: DM контекст ✅
- Монстры: "Монстр: Гоблин 1 (HP 7/7, AC 13, позиция 8,1) — описание"
- Промпт: точное имя цели из контекста
- Состояние: исправлено

### Баг 5: Урон ✅
- findMonsterByTargetName: fuzzy matching (case-insensitive, includes, startsWith)
- Логирование при отсутствии монстра
- Состояние: исправлено

### Баг 6: Английский/китайский ✅
- Промпт: "КАТЕГОРИЧЕСКИ ЗАПРЕЩЕНО использовать английские или китайские слова"
- Состояние: исправлено

### Баг 7: Переход хода ✅
- advanceExplorationTurn после каждого успешного действия
- Invalid действия не тратят ход
- Состояние: исправлено

### Баг 8: Friendly fire ✅
- findMonsterByTargetName проверяет совпадение с именем игрока
- Если target = игрок → friendlyFire=true, урон не применяется
- Состояние: исправлено

### Баг 9: UI масштаб ✅
- Класс ui-scale-* применяется к documentElement (<html>)
- CSS: html.ui-scale-* с правильной специфичностью
- Состояние: исправлено

### Баг 10: Лишние враги ✅
- При открытии боя раскрывается только целевой монстр
- Состояние: исправлено

### Баг 11: Быстрые применения ✅
- canQuickUse = !combatActive || isYourTurn
- В exploration все могут, в бою только текущий
- Состояние: исправлено

### Баг 12: Подсветка хода ✅
- currentTurnName = currentEntry?.combatantName ?? currentExplorerName
- В exploration подсвечивается текущий исследователь
- Состояние: исправлено

## Ячейки заклинаний (механика BG3):
- ✅ Ячейки видны в UI (BottomPanel + SpellbookPanel)
- ✅ Каждое заклинание требует ячейку (spendSpellSlot)
- ✅ Короткий отдых: восстанавливает HP (кость хитов) + ячейки варлока
- ✅ Долгий отдых: полное HP + ВСЕ ячейки + снятие коротких состояний
- ✅ Заговоры (cantrips) не требуют ячеек
- ✅ При отсутствии ячеек — action invalid
- ✅ DM промпт содержит инструкцию о трате ячеек
- ✅ Auto-upcast: если ячейка нужного круга кончилась, тратится более высокая
