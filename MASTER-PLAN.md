# DUSKFALL — Глобальный план доработок (BG3 + DOS2 + D&D 5e)

> Сопоставление механик Baldur's Gate 3, Divinity: Original Sin 2 и классических
> настольных D&D 5e с текущим состоянием проекта DUSKFALL.
> Создано после полного аудита кодовой базы (~10K строк DM-агента + 28 UI-компонентов + 19 Prisma-моделей).

## Текущее состояние (что уже есть)

### ✅ Реализовано и работает
- ИИ-Мастер (LLM planning → dice → state mutations → LLM narration, ~3000 строк)
- Сетка 16×16 с рельефом, укрытиями, высотами, AoE-оверлеями
- Создание персонажа: 12 классов × 9 рас × 10 происхождений + 126 талантов + 22 подкласса
- 34 заклинания (заговоры + L1-L5), ячейки L1-L9
- 51 монстр в бестиарии + 103 предмета (с редкостью, зачарованиями, проклятиями)
- Бой: инициатива, экономика действий (Action/Bonus/Reaction), спасброски смерти
- 18 состояний (poisoned, stunned, paralyzed, restrained, grappled, invisible, и т.д.)
- Очки движения (ОХ), Dash, Рывок, Схватка, Толчок, Two-Weapon Fighting
- Extra Attack для L5+ военных классов (бэкенд-обработка)
- Сопротивления/иммунитоны монстров, укрытия (cover AC), фланги, высота
- Креатив крафт (17 рецептов), алхимия, кузница, зачарование
- 6-язычная i18n (ru/en/es/de/fr/zh)
- Авторизация, 3 слота сохранений, бестиарий (разблокировка по обнаружению)
- Карта подземелья (BSP-генерация), переходы между комнатами, ловушки
- NPC с диспозицией, журнал квестов, память истории
- A* pathfinding для движения монстров, освещение (torch + darkvision)
- Легендарные действия для боссов (6 боссов × 3 действия)
- Аудио-SFX, TTS-озвучка Мастера, генерация сцен (AI-изображения)

### ⚠️ Частично реализовано
- Эффекты состояния: DoT-урон работает, но нет UI-индикаторов на токенах сетки для всех 18 состояний
- Ресурсы класса: тратятся, но нет UI-кнопок для их активации (только через чат-команды)
- Ловушки: модель существует, но нет UI для обнаружения/обезвреживания
- Скрытность: Скрытая атака Плута существует, но нет режима stealth/hide на сетке
- Крафт: рецепты есть, но UI требует доработки

### ❌ Отсутствует (сопоставление с BG3/DOS2/D&D 5e)
