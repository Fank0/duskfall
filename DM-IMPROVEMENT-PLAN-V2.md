# План: Улучшение Мастера Игры v2

## Что улучшаем:

### 1. Тактика монстров в промпте
- Добавить правила: цели слабых игроков, отступление при низком HP, использование способностей
- Фланги монстров, атака оглушённых целей

### 2. NPC отношения и память
- Сохранять изменения disposition NPC в story memory
- DM помнит кто помог/оскорбил NPC

### 3. Динамические DC с учётом уровня
- В промпте: DC должен зависеть от уровня группы и сложности задачи

### 4. Последствия провалов
- Добавить в промпт: провал проверки = реальное последствие (ранение, потеря предмета, ухудшение отношения NPC)
- В story memory сохранять последствия провалов

### 5. Нарративные отсылки
- В промпте нарратива: ссылаться на предысторию игрока, прошлые события
- Использовать имена NPC из памяти

### 6. Богатое вступление
- Улучшить generateUniqueIntro — больше деталей, NPC, hook сюжета
- Вступление должно включать: локацию, атмосферу, возможные пути, намёк на угрозу

### 7. Окружение в контексте
- Добавить описание комнаты подземелья (из MapRoom.description)
- Время суток влияет на доступные действия

### 8. Лог действий игроков
- Отслеживать что КАЖДЫЙ игрок делал в последние несколько ходов
- DM знает кто активен, кто пассивен

### 9. Масштабирование встреч
- В контексте: средний уровень группы
- В промпте: рекомендации по CR монстров

### 10. Улучшенный нарратив
- Нарратив должен включать: последствия, реакции окружения, эмоции
- Ссылки на прошлые события из памяти сюжета
