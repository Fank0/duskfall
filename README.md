# DUSKFALL

D&D 5e Virtual Tabletop с автономным AI Мастером Подземелий.

## Возможности

### Базовые
- 🎲 AI Dungeon Master (LLM-driven) с строгими правилами атмосферы
- 👥 Мультиплеер (комнаты, socket.io, ходы по инициативе)
- 🗡️ BG3-style создатель персонажа (12 классов, 9 рас, 10 фонов)
- 📊 Point-buy характеристики + 120 талантов (2-круговое дерево + ASI)
- ✨ Способности (расовые/классовые/свитки/заклинания)
- 📖 Предыстория + мировоззрение
- 🔐 Аккаунты + 3 слота сохранений (bcrypt + HMAC cookies)

### Боевая система 2.0
- 🤢 10 состояний (яд/оглушение/горение/...) с иконками на токенах
- ⬆️⬇️ Преимущество/Помеха (5 источников, отмена)
- 🔮 Спелл-слоты + короткий/долгий отдых
- 💥 AoE заклинания (круг/линия/конус, спасброски)
- ⚔️ Фланги и высокое положение

### Мир 2.0
- 📜 Журнал квестов (автоматическое извлечение)
- 🗺️ Карта мира (BSP-генерация, туман войны)
- 💬 NPC + диалоги (торговля, проверки характеристик)
- 🌅 Смена дня/ночи (4 фазы, тинты сцены)
- 🌧️ Погода (дождь/туман/гроза/снег, CSS-анимации)
- 🎲 Случайные события (6 типов)

### Прогрессия
- 🌳 Дерево навыков (2 круга + пререквизиты + ASI на 5/9/13/17)
- 🛡️ Экипировка (8 слотов, бонусы AC, ограничения по классу)
- 🔨 Крафт (17 рецептов, 3 верстака)

### Контент
- 👹 Бестиарий: 51 монстр (7 категорий, масштабирование по уровню)
- 📚 Книга заклинаний: 34 заклинания (8 школ, 6 кругов)
- 🎒 База предметов: 108 предметов (5 редкостей, сет-бонусы, артефакты с проклятиями)
- 🏰 Генератор подземелий: 5 биомов, BSP, ловушки, боссы, глубина

### Визуал/UI
- 🎬 Анимации боя (движение, удар, вспышки, screen shake)
- 🪙 Токены (портреты, рамки, HP-бар, ауры)
- 📊 Лог боя (фильтр, цвет, экспорт)
- ✨ Эффекты сетки (лут, ловушки, угрозы, AoE)
- 🎨 4 темы + 3 масштаба + сворачиваемые панели

### Аудио
- 🎵 Процедурная музыка (3 настроения: мир/бой/напряжение)
- 🔊 12 SFX (кость/удар/крит/промах/лечение/уровень/...)
- 🌧️ Погодный амбиент (дождь/гроза)
- 🎙️ TTS голосовой Мастер (3 голоса, авто-озвучка)

### Интернационализация
- 🇷🇺 Русский (основной)
- 🇬🇧 English
- 🇪🇸 Español
- 🇩🇪 Deutsch
- 🇫🇷 Français
- 🇨🇳 中文

### Технологии
- **Framework**: Next.js 16 (App Router, standalone output)
- **Language**: TypeScript 5
- **Database**: Prisma ORM + SQLite
- **Styling**: Tailwind CSS 4 + shadcn/ui
- **LLM**: мультипровайдерная цепочка (GLM → Gemini → OpenRouter → Ollama → z-ai-sdk)
- **Real-time**: socket.io (mini-service на порту 3003)
- **Auth**: bcrypt + HMAC-signed cookies

## Установка

### Локальная разработка
```bash
bun install
cp .env.example .env  # заполнить API ключами
bun run db:push
bun run dev
```

### Переменные окружения
```
GLM_API_KEY=...        # z.ai (основная, рекомендуется)
GEMINI_API_KEY=...     # Google Gemini (фолбэк 1)
OPENROUTER_API_KEY=... # OpenRouter: NVIDIA/Qwen3/Llama (фолбэк 2)
OLLAMA_BASE_URL=...    # локальная Ollama (фолбэк 3, опционально)
SESSION_SECRET=...     # 32+ символа для подписи cookies
DATABASE_URL=file:/data/custom.db
```

### Деплой на Railway
1. Загрузите код на GitHub
2. Подключите репозиторий в Railway
3. Настройте переменные окружения
4. Railway автоматически соберёт и запустит

## Стек LLM

Цепочка фолбэков (пробуются по порядку при сбое):
1. **GLM-4.6** (z.ai) — основная, лучший русский
2. **Gemini 2.0 Flash** (Google) — фолбэк 1
3. **OpenRouter**: Qwen3 → NVIDIA Nemotron → Llama 3.3 → GPT-OSS — фолбэк 2
4. **Ollama** (локальная) — фолбэк 3
5. **z-ai-web-dev-sdk** sandbox — последний резерв
